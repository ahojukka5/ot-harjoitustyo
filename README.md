# Saehaekkae - saehkoen saeaelimaetoen saeaestaejae

Saehaekkae (lue: "sähäkkä") on ohjelma, jonka tarkoituksena on pienentää
käyttäjän sähkölaskua. Sovellus on Helsingin yliopiston ohjelmistotekniikan
harjoitustyö.
